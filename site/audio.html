Аудиоплеер HTML5

Используется тег <audio> для воспроизведения звука.

Пример аудиофайла

[Здесь был бы аудиоплеер]

Для работы нужно добавить файл в папку audio/ и указать:

<audio controls>
  <source src="audio/sound.mp3" type="audio/mpeg">
</audio>

Форматы аудио

  • MP3 - самый популярный
  • WAV - без сжатия
  • OGG - открытый формат
